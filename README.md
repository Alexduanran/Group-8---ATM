# Group-8---ATM

To run:
    1. Open two terminals and cd into atm-flask and atm-react
    2. In the terminal of atm-flask, run:
        - export FLASK_APP=api
        - export DEBUG_MODE=1
        - flask run
    3. In the terminal of atm-react, run:
        - npm start
    4. The localhost will open in the browser
